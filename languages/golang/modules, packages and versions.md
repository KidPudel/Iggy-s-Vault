# Go Modules & Packages

https://go.dev/doc/modules/gomod-ref

**Package** — directory of `.go` files compiled together. Declared with `package name` at the top of each file.

**Module** — collection of packages versioned and distributed together. Defined by `go.mod`.

`go.mod` — module path, Go version, dependencies + versions
`go.sum` — dependency checksums (auto-generated)
